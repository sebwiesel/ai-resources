# AI Resources for Portfolio Managers

This repository is a shared knowledge base for portfolio managers and adjacent investment teams working with AI in a professional setting.

It combines:

- A practical guide to AI concepts, agents, workflows, and prompting
- Curated prompt and skill libraries with a review-oriented contribution model
- Worked examples and presentation materials as they become available
- A curated set of external resources for continued learning

## Working Principles

- Keep content practical, reviewable, and suitable for professional use.
- Do not commit client data, credentials, proprietary documents, or confidential notes.
- Treat prompts and skills as reusable assets, not one-off chat fragments.
- Prefer clarity over novelty. Content should be understandable by non-engineers.

## Contributions

Contributions are welcome, but prompts, skills, and examples should meet a higher review bar than general documentation. See [CONTRIBUTING.md](CONTRIBUTING.md) before opening a pull request.
