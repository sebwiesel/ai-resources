# Prompt Templates

Prompt templates are the lower-friction entry point for reusable AI workflows.

Use this area for lightweight, reviewable templates that do not yet justify a full skill package. Each prompt should make the intended user, required inputs, and expected output structure explicit.

Current categories:

- [Analysis](analysis/README.md)
- [Communication](communication/README.md)
- [Research](research/README.md)

If a prompt becomes operationally important, stable, or tool-dependent, it should likely be promoted into a reviewed skill under `skills/`.
